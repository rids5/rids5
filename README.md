## My Projects

---

### [Fraud Net](https://github.com/Swastika23214/Fraud_Net)
_A rule-based fraud detection system for telecom call data built in C++._

Features:
- Detects suspicious call patterns using frequency maps and hash maps
- Flags anomalies like unusually high call durations and repeated targets
- Processes large datasets efficiently using C++ STL containers

🔗 [View on GitHub](https://github.com/Swastika23214/Fraud_Net)

---

### [Healthcare Assistant](https://github.com/Swastika23214/Healthcare_Assistant)
_A Python CLI tool for daily health monitoring and reporting._

Features:
- Logs symptoms, meals, medications, and exercise daily
- Generates structured health reports using file I/O
- Clean data validation and user-friendly prompts

🔗 [View on GitHub](https://github.com/Swastika23214/Healthcare_Assistant)

---

### [Diwali Sales Data Analysis](https://github.com/rids5/Diwali-Sales-Data-Analysis)
_Exploratory data analysis on festive season sales using Python._

Features:
- Cleaned and processed raw sales data with Pandas
- Visualised buying trends across age, gender, state, and category
- Uncovered actionable business insights using Matplotlib & Seaborn
